# CUFF vs TIME Electrode Comparison in COMSOL

## Project Overview

This project focuses on the computational comparison between two neural stimulation electrode configurations:

* **CUFF electrodes**
* **TIME electrodes** (Transverse Intrafascicular Multichannel Electrodes)

The simulations are implemented in COMSOL Multiphysics in order to evaluate and compare the electrical stimulation behavior produced by the two electrode types when applied to peripheral nerve fascicles.

The main goal of the project is to investigate how the geometry and positioning of the electrodes influence:

* Electric potential distribution
* Selectivity of neural stimulation
* Fascicle recruitment
* Axonal activation

The project includes finite element simulations of the nerve environment together with neuronal activation analysis.

---

# Project Structure

The COMSOL models include:

* Peripheral nerve geometry
* Fascicles embedded inside the nerve
* Surrounding biological tissues
* Electrode geometries (CUFF and TIME)
* Electrical physics interfaces
* Meshing and solver configurations
* Axonal activation models

The repository is organized to support direct comparison between the CUFF and TIME electrode configurations.

---

# CUFF Electrode Model

The CUFF electrode configuration is modeled as an external electrode wrapped around the nerve.

Main characteristics:

* Extraneural stimulation approach
* 8 Electrodes positioned around the nerve surface
* Non-invasive interaction with the fascicles
* Distributed electric field generation

The COMSOL model includes:

* Nerve and fascicle geometries
* Conductive extracellular medium
* Electrode domains and boundaries
* Electrical insulation conditions
* Current injection settings
* Finite element mesh optimization

The CUFF model is used as a reference configuration for evaluating stimulation selectivity and field penetration.

# Axonal Model

Inside each nerve fiber, an axon is included in order to evaluate whether the fascicle is effectively stimulated.

The axonal behavior is modeled using the **Hodgkin-Huxley (HH) model**, which allows the simulation of:

* Membrane potential dynamics
* Action potential generation
* Ionic channel behavior
* Neural excitation thresholds

The HH-based implementation is used to determine whether the applied extracellular stimulation generated by the electrode produces successful neural activation.

This approach enables a more realistic validation of the stimulation effectiveness beyond the sole electric field analysis.

# Geometry Generation

The geometries used in the simulations are generated and imported through the integration between MATLAB and COMSOL Multiphysics.

MATLAB is used to:

* Define parametric geometries
* Generate nerve and fascicle structures
* Control geometric dimensions and positioning
* Automate model configuration
* Simplify the creation of multiple simulation scenarios

The generated geometrical data are then transferred to COMSOL through the MATLAB-COMSOL interface, enabling automated model construction and simulation workflows.

# Physics and Simulation Environment

The simulations are implemented using COMSOL Multiphysics.

The project mainly relies on:

* Electric Currents physics
* Conductive media modeling
* Finite Element Method (FEM)
* Time-dependent and stationary studies

The simulation workflow includes:

1. Geometry generation
2. Material assignment
3. Boundary condition definition
4. Mesh generation
5. Electrical stimulation setup
6. Solver execution
7. Axonal activation analysis

# Materials and Parameters

The models include biological tissues with different electrical properties.

Typical parameters involved in the simulations include:

* Tissue conductivity
* Electrode conductivity
* Fascicle dimensions
* Nerve diameter
* Electrode dimensions
* Current amplitude
* Pulse duration
* Mesh density

These parameters can be modified to analyze different stimulation conditions and electrode performances.
